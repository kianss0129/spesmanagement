# Threat Model

## System Components
- Web Client
- Application Server
- Database Server

## STRIDE Analysis
- Spoofing: Fake user login
- Tampering: Altered data
- Repudiation: Denial of actions
- Information Disclosure: Data leaks
- Denial of Service: Excess requests
- Elevation of Privilege: Role abuse

## Mitigations
- Strong authentication
- Authorization checks
- Logging and monitoring
